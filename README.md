# couchdb

[![Build](https://github.com/gsmlg-ci/couchdb/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/couchdb/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/couchdb`
- `ghcr.io/gsmlg-dev/couchdb`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/couchdb:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/couchdb:latest
```

## Build

```bash
docker build -t gsmlg/couchdb .
```
